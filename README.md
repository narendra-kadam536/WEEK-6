# Interactive Sales Dashboard

## Overview
This project analyzes sales data using Python to identify trends, product performance, and regional insights.

## Dataset
The dataset includes sales date, product, region, quantity, price, and total sales.

## Tools Used
- Python
- Pandas
- Seaborn
- Plotly

## Features
- Sales trend analysis
- Product-wise performance
- Regional sales distribution
- Correlation heatmap
- Interactive Plotly visualizations

## How to Run
pip install -r requirements.txt
python dashboard.py
